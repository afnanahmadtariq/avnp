Design Principles
One primary action per screen.
Motion should explain, never decorate.
Whitespace is a feature, not empty space.
Typography carries hierarchy—not color.
Every animation should reduce perceived waiting time.

If someone watches your demo without hearing a word, they should immediately understand the workflow and come away with one thought:

"That is incredibly smooth."
