# Relay brand assets

The production identity is **Relay**. The connected geometric `R` represents a customer brief moving through calls, offers, and a verified outcome.

## Production files

- `relay-mark.svg` is the square product mark for navigation, application icons, and compact placements.
- `relay-lockup.svg` combines the mark and Relay wordmark for horizontal placements.

The core palette is graphite `#101114`, warm ivory `#F5F4EF`, cobalt `#3157F6`, and signal mint `#86E6B3`. Keep clear space around the mark equal to one quarter of its width and do not recolor individual path nodes independently.

Archived, unapproved explorations remain in `concepts/` for traceability only.
